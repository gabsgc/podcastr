## :computer: Technologies

This project was made using the following technologies:
- React.JS
- Next.JS
- TypeScript

## ⛏ Dependencies
- Yarn
- JSON Server
- Axios
- Sass
- Data-fns
- Rc-slider

## :construction_worker: How to run

```
# Clone Repository
$ git clone https://github.com/gabsgc/podcastr

# Go to folder
$ cd podcastr

# Install Dependencies
$ yarn

# Run Web Server
$ yarn dev

# Run Server
$ yarn server
```
